# xkcdBot A simple discord which lets you see xkcd comics prefix = "xkcd " [Invite link](https://discord.com/api/oauth2/authorize?client_id=718079038471798824&permissions=0&scope=bot) ## Commands: - `xkcd random` Retrieves a random xkcd comic - `xkcd latest` Retrieves the latest xkcd comic - `xkcd comic [number]` Retrieves the corresponding xkcd comic to `[number]` - `xkcd random-whatif` This retrieves a random WhatIf link - `xkcd latest-whatif` This retrieves the latest WhatIf link - `xkcd whatif [number]` This retrieves the corresponding Whatif to `[number]` - `xkcd feedback [message]` This sends the developer(s) of the bot your message (cannot be over 2000 characters)
