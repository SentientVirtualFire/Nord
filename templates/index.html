xkcdBot

Bot Invite Link Server Invite Link

Discord bot with xkcd comics and WhatIfs at hand.

Note: this bot is not created or (as of writing) endorsed by xkcd or Randall Munroe.

Commands

xkcd help

description: Shows list of the commands and what they do.

xkcd random

description: Retrieves a random xkcd comic

xkcd latest

description: Retrieves the latest xkcd comic

xkcd comic [number]

description: Retrieves the corresponding xkcd comic to [number]

xkcd random-whatif

description: This retrieves a random WhatIf link

xkcd latest-whatif

description: This retrieves the latest WhatIf link

xkcd whatif [number]

description: This retrieves the corresponding Whatif to [number]

xkcd servers

description: This retrieves number of servers the bot is in and total number of users shared across the

xkcd feedback [message]

description: This sends the developer(s) of the bot your message (cannot be over 2000 characters)
